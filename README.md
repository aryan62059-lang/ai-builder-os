# AI Builder Roadmap Dashboard

A polished React + Vite + Tailwind + Framer Motion dashboard built from the existing roadmap component. It keeps the original XP, phases, tools, career, and first-week roadmap foundation, then upgrades it into a scalable product-style frontend.

## Folder structure

```txt
roadmap-dashboard/
  src/
    components/
      dashboard/
      layout/
      productivity/
      roadmap/
      ui/
    data/
    hooks/
    pages/
    utils/
```

## Run

```bash
npm install
npm run dev
```

Progress persists in `localStorage` under `ai-builder-roadmap-state-v2`.
